## ✨ Features

- 🏠 **Hero Section** — Animated typing intro with quick stats (projects built, technologies, dedication)
- 👨‍💻 **About Me** — Short bio, education, and current availability status
- 🛠️ **Skills Section** — Visual cards for HTML, CSS, JavaScript, Git, and GitHub with proficiency levels
- 📂 **Projects Showcase** — Highlighted projects with live demo & GitHub repo links
- 📩 **Contact Section** — Direct email, location, and social links (GitHub, LinkedIn, Facebook)
- 📄 **Download CV** — One-click resume download
- 📱 **Fully Responsive** — Optimized for both desktop and mobile devices

## 🚀 How It Works

1. On load — Hero section displays an animated typing effect introducing the developer
2. About section shares a short bio, education, and location
3. Skills section shows technologies known with proficiency badges
4. Projects section lists featured work with **Live Demo** and **GitHub** links
5. Contact section allows visitors to reach out via email or social platforms
6. Visitors can download the CV directly with one click
